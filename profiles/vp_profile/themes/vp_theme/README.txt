Site width is 1000px which is maximum width where horizontal scrollbar
will not show up on 1024 resolution screen. Tests showed that horizontal
scrollbar will appear on Windows 7 default theme at 1008px content width on IE,
Chrome and Firefox. It's a close call but will probably work on most desktops.

To compile SASS you also need "Sass Globbing Plugin"
( https://github.com/chriseppstein/sass-globbing )
